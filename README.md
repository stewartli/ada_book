## Github Action Setup


1. Initialize bookdown project.
   * Modify `_bookdown.yml`, `_output.yml`. 
   * Bookdown `build`.
   * `git init`, `.gitignore`. 
2. Create github repo. 
   * `git push`.
   * Set up github pages.
3. ~~`use_github_action("bookdown.yaml")`~~
4. ~~`renv init`~~
